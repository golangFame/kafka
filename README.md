# Kafka-Go-Consumer

A minimalistic Kafka go consumer for  local and cloud hosted kafka servers.

## Prerequisites

1. Kafka-Server 
2. Kafka-Producer

##Config

1. Kafka server 
    1. Topic 
    2. Hosted - localhost/SaaS (Software as Service)
2. Go 

## Usage

```bash
git clone https://github.com/goferHiro/kafka-go-consumer/
cd kafka-go-consumer
go run .

```
